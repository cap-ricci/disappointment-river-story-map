About

The webmap project Disappointment River was developed at the University of Trier as part of the Master's programme Applied Geoinformatics for the course Kartographisches Projektstudium II. The steps included the conception of the project idea, the structuring and delegation of tasks between the members of the team, the research of relevant topics and information, the coding of the web page and lastly the documentation and presentation of the results. The coding of the page was executed using HTML, CSS and Javascript. The inclusion of maps and geographic information was done using Geoserver and the Openlayers library. The full code can be found on Github.
